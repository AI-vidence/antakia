Theory behind AntakIA

Dimensionality Reduction

  • The projection in the Values space represents the 8-dimensional values from the dataset in 2D/3D.
  • The projection in the Explanation space is the 2D/3D representation of the Shapley value from the 8-dimensional dataset.
  • The Shapley value represents the deviation of each data point from the average.

Methods

  • UMAP creates a high-dimensional graph representation of the data and optimizes a low-dimensional graph to closely match its structural characteristics.
  • PCA is a method that transforms variables linked to each other (referred to as 'correlated' in statistics) into new variables that are decorrelated from each other.
  • t-SNE identifies similarities in the data and constructs a map where similar data points are positioned close to each other, while different ones are positioned farther apart.
  • PaCMAP is a technique for reducing dimensionality that can be applied to visualize data while preserving both the local and global structures present in the original dataset.
